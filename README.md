## ☕ CAYA — Business Analytics

## Objetivo
Analizar ventas, gastos y turnos para mejorar **margen** y **decisiones operativas** en una cafetería pick-and-go.

## Preguntas de negocio
- ¿Qué productos aportan más margen?
- ¿Qué días/horas venden mejor?
- ¿Qué gastos impactan más la utilidad?

## Datos
- Ventas (diario / mensual)
- Gastos diarios
- Inventario / insumos

## Proceso
- Limpieza y preparación de datos
- EDA (tendencias, outliers, patrones)
- KPIs (ventas, utilidad, margen, ticket promedio)
- Visualizaciones y recomendaciones

## Herramientas
Python | Pandas | SQL (opcional) | Excel/Sheets
